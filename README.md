## üìù About The Project

Expiravoid automizes your household by covering the whole process from managing groceries to generating shopping lists based on your planned meals.
This way, the web application not only simplifies your household but also helps in effectly reducing food waste.

Specifically, Expiravoid will feature the following core functionalities:
* Managing the groceries you currently have at home
* Creating / importing recipes and getting recommendations based on available groceries
* Creating a weekly meal planner based on recipes
* Automatically generating shopping lists based on meal planner as well as editing or manually creating them
